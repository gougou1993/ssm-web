# ssm-web
SpringMVC+Spring+MyBatis+Druid+PostgreSQL+AdminLTE+Maven+Motan框架集成 

##  介绍

### ssm-blank
空框架

### ssm-admin
带有基本权限管理模块

### ssm-motan-web
ssm-admin 基于motan剥离的前端

### ssm-motan-module
ssm-admin 基于motan剥离的服务端

### ssm-motan-api
ssm-admin-web/module接口、实体类、工具类

##  开发环境

```
jdk1.7.0_80
apache-maven-3.5.0
IntelliJ IDEA Community Edition 2017.1.4 x64
```
